Project Structure

Ryen King will work correctly only on the web server!

The structure of the "Ryen King. A digital designer HTML Template":

  • css - all common scripts used on the pages
  • favicons - all favicons, favicons were generated here https://www.favicon-generator.org/
  • fonts - all fonts used on the pages
  • images - images, icons and backgrounds for all secondary pages of the project
  • img - images, icons and backgrounds for the main page of the project
  • js - all common scripts used on the pages file

Swiper Slider

All sliders are using "Swiper slider". It works with Swiper jQuery plugin. It's easy to customize, see documentation

Home Parallax Piling

Home parallax piling works with jQuery PagePiling plugin

Experience cursors

To change a cursor on a specific section in the experience block, you need to save your new cursor in .png format into img > main > experience folder and set its path in a data-cursor attribute
